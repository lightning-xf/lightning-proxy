# Lightning Proxy 发布说明

## v1.0.0 - 首次发布

### 发布日期
2024 年

### 平台支持
- **Android** 7.0+
- **Windows** 10/11

### 主要功能

#### Android 版本
- 一键启动/停止 VPN 代理
- 多协议支持：VLESS (Reality), VMess, Trojan, Shadowsocks, Hysteria2, TUIC
- 智能分流规则
- 节点延迟测试
- 订阅支持
- 多语言（中文/英文）

#### Windows 版本
- 系统代理自动配置
- 深度休眠模式，低资源占用
- 托盘快速操作
- 开机自启
- 单实例运行
- 多协议支持

### 核心特性
- 基于 Xray 核心
- 高性能网络传输
- 智能路由分流
- 用户友好的 UI

### 下载地址
请前往 [GitHub Releases](https://github.com/lightning-xf/lightning-proxy/releases) 下载最新版本。

### 安装说明

#### Android
1. 下载 APK 文件
2. 在设备上启用“未知来源”安装
3. 安装 APK
4. 授予 VPN 权限

#### Windows
1. 下载安装程序
2. 以管理员身份运行
3. 按照安装向导完成安装
4. 首次运行时授予管理员权限

### 更新日志

#### v1.0.0
- 初始版本发布
- 支持 Android 和 Windows 双平台
- 集成 Xray 核心
- 基础代理功能
